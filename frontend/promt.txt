Build a single-page frontend application using **React.js** (you may use React Router for navigation between views).

Do NOT invent new pages or features. Use ONLY the following three levels:

1) Home page (image 1 layout + image 2 children in the center)
2) Region page (layout exactly like image 3)
3) Region services page (layout exactly like image 4)

Use a dark galaxy background and golden decorative ornaments for ALL pages, the same as the home page background.

====================================
GLOBAL STYLE
====================================

- Background: dark blue / dark space gradient with stars and glowing golden particles, identical on all screens.
- Corners: golden Uzbek-style ornaments in all four corners (top-left, top-right, bottom-left, bottom-right), same shape as in the reference images.
- Main color: gold / orange for frames, headings and buttons.
- Typography:
  - Page titles: serif font (similar to the ones in the designs).
  - Text: clean sans-serif.
- Buttons:
  - Rounded rectangle with gold/orange background and white text.
  - Hover: slightly brighter with soft glow.
- Use static demo data (arrays in JS) instead of API calls.

====================================
APP STRUCTURE
====================================

Create the following React components / views:

1. `HomePage`
2. `RegionPage`
3. `RegionServicesPage`

Use React Router or a simple state machine so:

- Home → Region
- Region → RegionServices
